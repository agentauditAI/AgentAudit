How It Works

Four steps to full compliance.

From agent registration to on-chain proof — AgentAudit handles the full compliance lifecycle.

01
Register Agent
Call AgentRegistration.sol with agent type, framework, and owner address. Agent gets an on-chain identity.
02
Log Actions
SDK intercepts agent actions and submits audit batches to AuditVault. Every action timestamped on Mantle.
03
Score & Monitor
Compliance score calculated per batch. Dashboard shows real-time risk status across all registered agents.
04
Certify
Export EU AI Act compliance certificate. Signed by a licensed legal professional. Ready for regulators.
