// Compliance

Built for EU AI Act.

High-risk AI systems must maintain logs. AgentAudit makes compliance automatic.

// Install
npm install @agentaudit/sdk

// Usage
import { AgentAudit } from '@agentaudit/sdk';

const audit = new AgentAudit({
  contractAddress: '0x...',
  rpcUrl: 'https://rpc.sepolia.mantle.xyz',
  privateKey: process.env.AGENT_PRIVATE_KEY,
});

// Log an action — returns txHash + logIndex
const receipt = await audit.log({
  action: 'TRANSFER_APPROVED',
  metadata: { amount: '100 USDC', recipient: '0x...' },
});

console.log(receipt.txHash); // on-chain proof ✅

Six layers of on-chain intelligence.

⚙️

MCP Server

Plug any MCP-compatible agent in minutes. No custom SDK. register_agent, commit_audit_batch, get_risk_score — exposed over stdio transport and ready to use.

mcp-server.ts
🪪

ERC-8004 Agent Identity

On-chain Know Your Agent: name, version, developer wallet, capabilities hash, and compliance tier. Immutable, queryable, cross-validated against AgentRegistration.

AgentIdentityRegistry.sol
🧠

Decision Provenance

Log WHY the agent acted — model version, input data hash, active policy, and trigger event. Causal chain anchored on-chain without exposing raw prompts.

commitBatchWithProvenance()
⚠️

Risk Scoring

Every batch scored HIGH / MEDIUM / LOW automatically. TRANSFER, BRIDGE, DRAIN → HIGH. SWAP, STAKE → MEDIUM. Action type and spend thresholds enforced on-chain.

AuditVault.sol · _computeRisk()
📊

EU AI Act Auto-Reports

generateComplianceReport() reads all on-chain data and produces a structured COMPLIANT / NEEDS_REVIEW / NON_COMPLIANT verdict with full audit summary. Zero manual work.

EUAIActReporter.sol
